# Marketing Campaign EDA

## Overview
Performed Exploratory Data Analysis (EDA) on a marketing campaign dataset to evaluate channel performance and recommend budget allocation based on ROI.

## Highlights
- Cleaned and analyzed campaign data
- Visualized marketing funnel and channel performance
- Compared ROI, ROAS, CTR, CPL, and Conversion Rates
- Identified top and low-performing marketing channels

## Recommendation
Increase budget for **Email, Paid Search, and SEO**, while reducing spend on **Influencer** and **Display Ads** due to low ROI.

## Tools Used
- Python
- Pandas
- Matplotlib
